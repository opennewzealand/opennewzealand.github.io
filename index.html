Code for NZ

Civic hacking for New Zealand.

What is Code for New Zealand? It's a Code for All Brigade for passionate citizens, designers, policy wonks, data scientists, and, well, anyone to code for their community. Brigade chapters, in collaboration with their governments, are using technology to enhance their communities — and you can too.

Send an email to join@codefor.org.nz to request access to the Code for NZ channel, and find out how you can help!
